# <ins>**Python-Installation**</ins>
## **AIM**<br>
To study the basics of Python programming language, understand the Python execution environment, install Python and Jupyter Notebook using Anaconda distribution, and explore various Python IDEs including Google Colab.<br>
## **OBJECTIVES**<br>
•	To understand Python features and applications<br>
•	To install Python and Jupyter Notebook using Anaconda<br>
•	To explore Python IDEs and environments<br>
•	To execute basic Python programs using Jupyter Notebook and Google Colab<br>
## **THEORY**<br>
### **1. Introduction to Python**<br>
Python is a high-level, interpreted, and object-oriented programming language widely used for data analysis, machine learning, automation, and scientific computing.<br>
### **2. What is Anaconda?** <br>
Anaconda is a free, open-source distribution of Python and R designed for data science and machine learning.<br>
#### **Advantages of Anaconda:** <br>
•	Comes with Python, Jupyter Notebook, Spyder, and libraries pre-installed<br>
•	Easy package and environment management<br>
•	Avoids dependency issues<br>
•	Suitable for beginners and professionals<br>
